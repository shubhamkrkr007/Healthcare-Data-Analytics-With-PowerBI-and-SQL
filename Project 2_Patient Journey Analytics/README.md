# Healthcare Patient Journey Analytics

## Description
This project analyzes patient treatment journeys using SQL and Power BI, focusing on therapy initiation, adherence, drop-off behavior, and provider/payer impact.

The goal is to derive actionable healthcare insights that can improve patient outcomes and treatment effectiveness.

## Key Insights
- Identified significant drop-off rates in patient treatment lifecycle
- Analyzed drug-level adherence and switching behavior
- Evaluated provider and hospital influence on patient outcomes
- Assessed payer impact on treatment adherence
- Measured treatment delays between diagnosis and therapy initiation

## Tools Used
- SQL (MySQL)
- Power BI
- Excel / CSV
